# FinalLeague

AI-powered football analysis project using computer vision and deep learning techniques for player/team recognition and match analysis.

## Overview
This project combines YOLOv8 object detection and face recognition models to analyze football match footage and identify players or teams from video input.

The system was developed as part of an AI League project and focuses on applying computer vision techniques to sports analytics.

---

## Features
- YOLOv8-based player detection
- Face recognition integration
- Football match video analysis
- Team/player identification
- Real-time computer vision pipeline
- Deep learning-based detection workflow

---

## Technologies Used
- Python
- YOLOv8
- OpenCV
- NumPy
- TensorFlow / Keras
- Deep Learning
- Computer Vision
